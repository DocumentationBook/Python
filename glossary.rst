Glossary
########

.. glossary::
   :sorted:

   Doc-SIG
      `Python Documentation Special Interest Group <https://www.python.org/community/sigs/current/doc-sig/>`_
      is a group of people responsible for discussion of both the form and content of Python documentation.

   PSF
      `Python Software Foundation <https://www.python.org/psf-landing/>`_ is an organization devoted to advancing open
      source technology related to the Python programming language.
